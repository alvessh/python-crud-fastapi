# API CRUD utilizando FastAPI

O projeto aprensenta as seguintes rotas:
# RAIZ
- [GET] /

# TIMES
- [GET] /game/
- [POST] /game/ 
- [PUT] /game/{id}
- [DELETE] /game/{id}

# GAMES
- [GET] /
- [GET] /team/
- [POST] /team/ 
- [PUT] /team/{id}
- [DELETE] /team/{id}

# PARA ACESSAR A DOCUMENTAÇÃO
- [GET] /docs
